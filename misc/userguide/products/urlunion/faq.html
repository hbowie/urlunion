PSTextMerge FAQ

Frequently Asked Questions

What the heck does this thing do?

PSTextMerge can do lots of things with rows and columns of data. The program can read such data in tab-delimited format, which is a common format that many database, spreadsheet and address book programs can both read and write. The program can also read tabular data from an MS Excel spreadsheet. PSTextMerge can read such data, sort it, filter it, and then generate Web pages from the data. It can also read data from HTML pages, in any of several different formats. It can also combine multiple files of similar data, intelligently merging the contents of records with duplicate keys. It can also read file directories, and generate Web pages from that data too.
