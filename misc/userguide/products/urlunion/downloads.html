Final Release Version 2.4

The final release of PSTextMerge version 2.4 is available for downloading. See our Version History page for information about the new release. The following download options are available.

InstallAnywhere Installers
Installers are available for Windows and most major Unix Platforms.
Mac OS X Disk Image
Select this option for the most complete Mac OS X experience. Simply download this file, mount it using Disk Copy, and drag the enclosed folder to your Applications folder. If you have already registered a previous version, simply drag the "parms.txt" file from the old PSTextMerge folder to the new one.
