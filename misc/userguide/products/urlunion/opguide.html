PSTextMerge Operating Guide

PSTextMerge presents the user with a series of tabs. Each of the tabs performs a different function. They will be described as they appear on the screen, going from left to right. This is also the sequence in which they would normally be used. Click on the name of a tab below for a complete description of the functions on that tab.

Starting with release 2.3, many of the file-related functions are now also available from menus and from shortcut keys. Pull down the menus to see the shortcut keys that are available. The command key is used in conjunction with another key to invoke shortcuts on a Mac, while the control key is used on other platforms.
