PSTextMerge Workshop

Let's put PSTextMerge through its paces and see what it can do.

Web Page Publication

PSTextMerge can be used to create and maintain a static Web page by merging a tab-delimited file into a specially formatted Web page template. See examples 1, 2, 3, 3a and 6.

File Management

PSTextMerge can read a file directory and then generate a Web page with data about the files. See example 4.

Eliminating Address Book Duplicates

Do you have an address book with duplicate entries? Or multiple address books, with some entries that are duplicated between the two? Save them as tab-delimited files, merge them into a single file using PSTextMerge, and use the Sort and Combine options to find and eliminate duplicates. Output the file from PSTextMerge and then Import it back into your address book manager. See example 7.

Bookmark Management

Do you have valuable URLs spread throughout multiple Web pages, Bookmark files and/or Favorites collections? Most bookmark managers and Web browsers will let you export URLs into an HTML file. PSTextMerge can then read these files as input, preserving folder structures as Categories, merging them into one list and eliminating duplicate entries. Then turn around and generate static Web pages from your master list in any of several popular formats. See example 5.
