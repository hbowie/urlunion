PSTextMerge

Our flagship product merges tabular data into text templates to create output text files: fully populated Web pages, XML files, RSS files, etc. Input data can be read from tab-delimited files, MS Excel spreadsheets, HTML pages and XML files. You can record and easily play back recorded scripts, making it easy to re-generate your output when your data changes. Tab-delimited files can be easily exported from most spreadsheets, databases, address book, and many other programs.

PSTextMerge can also perform other operations with tabular data, extracting and merging data from multiple bookmarks files, address books, and other formats.

Operational Overview

PSTextMerge can perform all of the following functions with lists of input data.

  • Input — Read a file in and prepare for further processing, optionally converting or cleaning up the data. In addition to reading tab-delimited data, PSTextMerge can read MS Excel spreadsheets, parse HTML (most notably bookmark files) and read file directories.
  • View — View the input data in a tabular format.
  • Sort — Arrange the records in the file based on any combination of key fields. Once sorted, records with duplicate keys can be intelligently combined, minimizing or eliminating data loss.
  • Filter — Show only selected records, based on one or more selection criteria.
  • Template — Create an output text file (such as an HTML file) by merging the tab-delimited data with a template file. One or more Web pages can be created from a single data source.
  • Output — Write the processed data back out to a new tab-delimited text disk file.
  • Script — Record and playback sequences of any of the above actions.
