PSTextMerge

PSTextMerge merges lists of data into template files to create output text files. The input data source may be a tab-delimited data file or a Microsoft Excel spreadsheet. The output text file may be an HTML Web page, an XML page, or an RSS feed, to give a few popular examples. You can record and easily play back recorded scripts, making it easy to re-generate your output text files when your data changes. It should be noted that tab-delimited files can be easily exported from most spreadsheets, databases, address book, and many other programs. PSTextMerge can also perform other operations with tabular data, extracting and merging data from multiple bookmarks files, address books, and other formats.
